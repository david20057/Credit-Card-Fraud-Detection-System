# Credit Card Fraud Detection System

## Overview

A machine learning project developed as part of a team of four to detect fraudulent credit card transactions using a real-world transactional dataset from Kaggle. The project focused on analysing highly imbalanced financial data, identifying fraud patterns, and evaluating multiple classification models to improve fraud detection performance.

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Jupyter Notebook

## Key Features

- Performed Exploratory Data Analysis (EDA) to identify fraud patterns and transaction anomalies.
- Applied data preprocessing and feature engineering techniques to prepare the dataset for modelling.
- Trained and evaluated multiple machine learning classification models.
- Compared model performance using metrics such as Precision, Recall, F1-Score, and ROC-AUC.
- Investigated the challenges of class imbalance in fraud detection datasets.

## Dataset

- Credit Card Fraud Detection Dataset (Kaggle)
- 284,807 transactions with highly imbalanced fraud classes.

## Results

The project demonstrated the effectiveness of machine learning techniques for identifying fraudulent financial transactions while highlighting the importance of using appropriate evaluation metrics when working with imbalanced datasets.
