# Credit-Card-Fraud-Detection-System

A machine learning project developed as part of a team to identify fraudulent financial transactions using real-world transactional datasets from Kaggle. The project involved exploratory data analysis, feature engineering, data preprocessing, and the evaluation of multiple classification models for fraud detection.

Technologies Used
1. Python
2. Scikit-learn
3. Pandas
4. NumPy
5. Machine Learning

Key Features
1. Exploratory Data Analysis (EDA)
2. Feature Engineering
3. Fraud Classification Models
4. Performance Evaluation on Imbalanced Data
